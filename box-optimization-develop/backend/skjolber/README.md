# Library

This folder makes use of the [3d-bin-container-library](https://github.com/skjolber/3d-bin-container-packing) by Skjolber.

You can find detailed information about the library and its usage [here](https://github.com/skjolber/3d-bin-container-packing)

## Files

`src/App.java` contains an example of a problem being solved.

`src/SkjolberSolver.java` contains the implementation used by the backend. 
- Creates Skjolber objects from standard java objects 
- Defines a solve function
